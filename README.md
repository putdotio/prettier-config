## Installation

```bash
pnpm add prettier @putdotio/prettier-config -D
```

## Usage

Add [`@putdotio/prettier-config`](https://github.com/putdotio/prettier-config) to your `package.json`:

```json
{
  "prettier": "@putdotio/prettier-config"
}
```
